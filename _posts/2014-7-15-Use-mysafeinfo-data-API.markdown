###### All datasets can be retrieved via the web service at

`http://mysafeinfo.com/api/data?list={listname}`

###### [List categories](https://mysafeinfo.com/content/categories)

[http://mysafeinfo.com/api/data?list=popular&format=json](http://mysafeinfo.com/api/data?list=popular&format=json)

[http://mysafeinfo.com/api/data?list=englishmonarchs](http://mysafeinfo.com/api/data?list=englishmonarchs)

[http://mysafeinfo.com/api/data?list=alcoholicbeverages&format=json](http://mysafeinfo.com/api/data?list=alcoholicbeverages&format=json)

[http://mysafeinfo.com/api/data?list=humor_quotes](http://mysafeinfo.com/api/data?list=humor_quotes)

###### Use the format parameter to request data in a specific format

[http://mysafeinfo.com/api/data?list=states&format=xml](http://mysafeinfo.com/api/data?list=states&format=xml)

[http://mysafeinfo.com/api/data?list=states&format=json](http://mysafeinfo.com/api/data?list=states&format=json)

[http://mysafeinfo.com/api/data?list=states&format=jsonp](http://mysafeinfo.com/api/data?list=states&format=jsonp)

[http://mysafeinfo.com/api/data?list=states&format=csv](http://mysafeinfo.com/api/data?list=states&format=csv)

[http://mysafeinfo.com/api/data?list=states&format=html](http://mysafeinfo.com/api/data?list=states&format=html)

###### Use the random parameter to randomize the order of the data returned

[http://mysafeinfo.com/api/data?list=humor_quotes&format=json&random=true](http://mysafeinfo.com/api/data?list=humor_quotes&format=json&random=true)

###### Use the rows parameter to control how many records are returned

[http://mysafeinfo.com/api/data?list=humor_quotes&format=json&random=true&rows=3](http://mysafeinfo.com/api/data?list=humor_quotes&format=json&random=true&rows=3)
